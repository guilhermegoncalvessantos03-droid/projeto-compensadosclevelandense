# 🌲 Compensados Clevelandense — Da Floresta ao Mundo

Uma Landing Page institucional, moderna e totalmente responsiva desenvolvida para a **Compensados Clevelandense**. O projeto destaca a jornada sustentável da empresa, desde o plantio controlado de pinus até a exportação de compensados de alta performance para mais de 30 países.

---

## 🚀 Demonstração

O projeto conta com uma interface limpa, transições suaves e componentes interativos (como carrossel automatizado e cards expansíveis ao passar o mouse) projetados para oferecer uma excelente experiência ao usuário.

---

## 🛠️ Tecnologias Utilizadas

A página foi construída utilizando tecnologias web nativas, priorizando a performance e a leveza do carregamento:

* **HTML5:** Estruturação semântica e acessível (uso de tags como `header`, `section`, `footer` e atributos `aria-label`).
* **CSS3 (Moderno):**
    * **Custom Properties (Variáveis CSS):** Centralização de paleta de cores, sombras e border-radius para fácil manutenção.
    * **Flexbox & CSS Grid:** Layouts complexos e alinhamentos precisos de forma nativa.
    * **CSS Animations:** Carrossel animado (`@keyframes`) rodando diretamente pela GPU de forma fluida.
    * **Design Responsivo:** Uso de `clamp()` para tipografia fluida e *Media Queries* sob medida para smartphones e tablets.
* **Google Fonts:** Fontes *Montserrat* (títulos de forte impacto) e *Poppins* (leitura leve e moderna).
* **Unsplash:** Imagens de alta definição otimizadas para o setor florestal e industrial.

---

## ✨ Funcionalidades e Diferenciais

* **Menu Fixo com Backdrop Filter:** Navegação que acompanha a rolagem da página com efeito de vidro fosco (*blur*).
* **Carrossel de Imagens 100% CSS:** Apresentação dos pilares da empresa sem a necessidade de bibliotecas JavaScript pesadas.
* **Linha do Tempo Interativa (`#jornada`):** Demonstração do ciclo de produção ("Do plantio à entrega global") com efeito *hover reveal*.
* **Cards de Sustentabilidade Resilientes:** Layout adaptável que se expande para revelar informações detalhadas ao interagir.
* **Seção de Métricas Impactante:** Exibição de dados estatísticos (como os mais de 10 mil hectares plantados) usando Grid responsivo.
* **Acessibilidade Focal:** Suporte para navegação via teclado (`tabindex`) nos cards informativos.

## 📦 Como Executar o Projeto

Como a aplicação é estática e utiliza apenas HTML e CSS nativos, você não precisa instalar nenhuma dependência (como Node.js ou pacotes npm).

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd nome-do-repositorio
    ```
3.  **Abra o arquivo no navegador:**
    Basta dar um duplo clique no arquivo `CompensadosClevelandense.html` ou utilizar a extensão **Live Server** no VS Code para visualizar com recarregamento em tempo real.
